# web_todo_app

A simple Flask-based Todo application that allows you to add, view, and remove todo items.

## Requirements

- Python 3.x
- Flask

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/web_todo_app.git
   cd web_todo_app
   ```

2. Create a virtual environment (optional but recommended):
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install Flask:
   ```bash
   pip install flask
   ```
